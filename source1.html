How to View the Source of This Page

If you go to your web browser's View menu, you will see an option that allows you to see the source code behind a web page.
When the web was first taking off this was a very popular way for people to learn how web pages had been built.
Today, people still often view the source of pages to learn how a web page has been built.

The Main Languages of the Web

HTML is the standard markup language for creating Web pages. HTML describes the structure of a Web page, and consists of a series of elements. HTML elements tell the browser how to display the content.

CSS is a language that describes how HTML elements are to be displayed on screen, paper, or in other media. CSS saves a lot of work, because it can control the layout of multiple web pages all at once.

JavaScript is the programming language of HTML and the Web. JavaScript can change HTML content and attribute values. JavaScript can change CSS. JavaScript can hide and show HTML elements, and more.

Scrambled Eggs

Eggs are one of my favorite foods. Here is a recipe for deliciously rich scrambled eggs.

Ingredients

  • 2 Eggs
  • 1 tbs Butter
  • 2 tbs Cream

Method

  1. Melt butter in a frying pan over a medium heat
  2. Gently mix the eggs and cream in a bowl
  3. Once butter has melted, add cream and eggs
  4. Using a spatula fold the eggs from the edge of the pan to the center every 20 seconds (as if you are making an omelette
  5. When the eggs are still moist remove from the heat (it will continue to cook on the plate until served
Atom

Atoms are the basic units of matter and the defining structure of elements. The term "atom" comes from the Greek word for indivisible, because it was once thought that atoms were the smallest things in the universe and could not be divided. ... Atoms were created after the Big Bang 13.7 billion years ago.

Molecule

Molecule refers to a group of atoms bonded together, representing the smallest fundamental unit of a chemical compound that can take part in a chemical reaction.

Ion

Ion is any atom or group of atoms that bears one or more positive or negative electrical charges.
Positively charged ions are called cations; negatively charged ions, anions.

Radio Button and Checkbox

Radio buttons allows users to pick just one of a number of provided options from the list.
With this type "radio" you cannot select and unselect, you only have one option and the user can only select another option.

Checkbox on the other hand is used if you are only allowing the user one option and want them to be able to diselect it
(for example if they are indicating they agree to terms and conditions), you should use a checkbox instead.
